## File Requirements:
- Format: .jpeg, .jpg, .png, or .webp
- All filenames must be lowercase
- No spaces in filenames
- Exact naming as listed above

## Troubleshooting:
1. Check browser console (F12) for image load errors
2. Verify all 12 files exist in this folder
3. Ensure file extensions match (.jpeg not .jpg if using .jpeg)
4. Check file permissions (files should be readable)
5. File size: Recommended under 2MB each for faster loading
